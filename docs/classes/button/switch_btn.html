SwitchBtnElement¶

SpWMLによる記法¶

(switchBtn)

解説¶

子ブロックを持たないブロックです。 このブロックを配置した場所には、ON・OFFスイッチが設置されます。

機能（Dart）¶

setCallback¶

スイッチがタップされたときのコールバックを設定します。 コールバック関数の引数にはボタンの状態が渡されます。

void setCallback(void Function(bool btnEnabled)? callback)

setSwitchValue¶

スイッチの値を設定します。 画面更新が必須であり、親ウィジェット内で引数を管理する必要があります。

void setSwitchValue(bool v)

変数（Dart）¶

  • 共通 変数。

  • [elParams] : Flutterの Switchクラス 相当のパラメータを持つ変数です。

利用可能パラメータ¶

  • 共通 パラメータ。

  • color (String, #AARRGGBB)。色の指定。
