PopupMenuBtnElement¶

SpWMLによる記法¶

(popupMenuBtn)
+(menu)選択肢1
+(menu)選択肢2
+(menu)選択肢3

解説¶

ポップアップメニューを作成できるブロックです。

機能（Dart）¶

子ブロックの操作¶

親クラス の機能。

コールバックの設定¶

コールバックを設定します。 メニューが選択された時に、そのインデックスが渡されます。

void setCallback(void Function(int index)? callback)

変数（Dart）¶

  • 共通 変数。

  • 親 変数。

  • [elParams] : Flutterの PopupMenuButtonクラス の使い勝手を向上するために調整を加えた変数を持っています。

利用可能パラメータ¶

  • 共通 パラメータ。

  • 親 パラメータ。

  • iconNum (String)。アイコンの種類を codePoint で指定します。

  • iconSize (double)。アイコンの大きさをpxで指定します。

  • iconColor (String)。色の指定です。#AARRGGBB表記、あるいは 2014 Material Design color palettes の色をred[500]やredAccent[100]のような表記で扱えます。
