エディタを使ってみよう¶

SpWMLで書いた簡易エディタがあるので、ぜひ色々試してみてください。

これはテスト表示用の簡素な作りになっているので、 他のコードエディタで作ったコードを貼り付けてUIをテスト・微調整するという使い方が一番良いと思います。

https://simple-widget-markup-editor.web.app/

SpWMLの記法について¶

SpWMLは、以下の Simple Block Markup Laungage(SBML) の派生形としての構造を持ちます。

(型, パラメータ1:値1, パラメータ2:値2,...)コンテンツテキスト

SBMLがXMLとするとSpWMLはHTMLのようなもので、SBMLの方がより自由度が高い言語です。 書き方についてはSBMLと同様で、型名などにtextやh1など、HTMLに近いものが事前に定義してあります。

例えばヘッダー1の大きさでテキストを記述したい場合、以下のようにします。

(h1)テキスト内容

型によって利用できるパラメータは異なります。 詳しい型名や利用できるパラメータなどはこのホームページの 「クラスと機能」 を参照してください。
