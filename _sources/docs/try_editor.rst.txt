SpWMLの記法について
====================
SpWMLは、以下のSimple Block Markup Laungage(SBML)の派生形としての構造を持ちます。

https://github.com/MasahideMori-SimpleAppli/simple_block_markup_language/blob/main/README_JA.md

SBMLがXMLとするとSpWMLはHTMLのようなもので、SBMLの方がより自由度が高い言語です。
書き方についてはSBMLと同様で、型名などにtextやh1など、HTMLに近いものが事前に定義してあります。

詳しい型名やパラメータなどはこのホームページの :ref:`「クラスと機能」<classes_and_methods>` を参照してください。
