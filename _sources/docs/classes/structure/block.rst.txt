.. _block:
======================================
BlockElement
======================================

SpWMLによる記法
===============

.. code-block:: none
   
   (block)
   +(単一の子要素)

解説
===============
一つだけの子要素を持つ領域区切り用の要素です。

機能（Dart）
===============
外部から利用するものはありません。内部的に使用されるメソッドのみを持ちます。

変数（Dart）
===============
* :ref:`共通<spwml>` 変数
* [child] : 単一の子ウィジェットです。
