.. _block:
======================================
BlockElement
======================================

SpWMLによる記法
===============

.. code-block:: none
   
   (block)
   +(単一の子ブロック)

----

解説
===============
一つだけの子ブロックを持つ領域区切り用のブロックです。

----

機能（Dart）
===============
外部から利用するものはありません。内部的に使用されるメソッドのみを持ちます。

----

変数（Dart）
===============
* :ref:`共通<spwml>` 変数
* [child] : 単一の子ウィジェットです。

==================
